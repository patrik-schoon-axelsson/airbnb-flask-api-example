Flask API sample app

This app is a sample API. You can find the github reposity for it here.

Endpoint Methods Input type Response
"/" "GET", "POST" GET: curl / browser
POST: "application/json" & object containing "name", "listing_url" and "description" keys 		GET response: index.html landing page
POST response: JSON document of newly added document
"/api/<query-string>" "GET" query string, e.g: "/api/?page_no=2&docs=10" GET response: Paginated database entries, where "page_no" equals off-set from beginning of collection and "docs" equals number of documents per query
"/api/listings/<document_id_string>" "GET", "POST", "DELETE" GET: document_id_string URL-variable, matching the "_id" field of a database document
POST: document_id_string URL-variable & application/json object, containing "name", "listing_url" and "description" keys.
DELETE: document_id_string URL-variable matching the "_id" field of a database document 		GET response: jsonified document-data matching document_id_string
POST response: json object confirming the update was succesful.
DELETE response: json object confirming deletion was succesful
